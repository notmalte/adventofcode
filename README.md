# Advent of Code

My solutions to the [Advent of Code](https://adventofcode.com/) puzzles in Rust.

Place your input files in the `inputs` directory, e.g. `inputs/2023/01.txt`.

Run the solutions like this:

```sh
cargo run -- --year <year> --day <day>
```
